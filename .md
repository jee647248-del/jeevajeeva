# Welcome to Jeeva — Demo Page

This small page demonstrates an animated nightscape with a blue-to-red animated background, a moon, twinkling stars, and floating symbols.

Files:
- `index.html` — the page. Open this in your browser to view the animation.

How to run locally (Windows):
1. Open File Explorer and navigate to the folder:
   `C:\Users\jeeev\docker.portfolio`
2. Double-click `index.html` or right-click -> Open with -> choose your browser (Chrome/Edge/Firefox).

Notes:
- No build or server is required; this is a standalone HTML file.
- If you want to serve it via a local server (optional), you can run a simple HTTP server. For example, with Python 3 installed run in PowerShell:

  python -m http.server 8000

  Then open http://localhost:8000/index.html in your browser.
